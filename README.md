# Executando o projeto

```bash
# Clonar o repositório
$ git clone https://github.com/joelmarcosgo/extrato_bancario_frontend.git

# Executar os comandos
$ yarn
# ou
$ npm install

# Rodar aplicação em modo de desenvolvimento
$ yarn start
# ou
$ npm start
```
